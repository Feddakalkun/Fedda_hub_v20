# FEDDA Hub v20

Workflow-first local AI studio: React frontend → FastAPI backend → ComfyUI.

v20 is a **UI and content reset** on top of the v19 installer/runtime foundation.

## Quick start (development)

```bat
cd app\scripts
install.bat LITE
cd ..
run.bat
```

Opens Vite on `http://127.0.0.1:5173`.
